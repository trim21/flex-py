# flex-bin

Prebuilt [flex](https://github.com/westes/flex) binary shipped as a Python wheel.

## Installation
```
pip install flex-bin
```

## Usage
```
flex --version
```

## Building locally
```
uv build --wheel .
```
Wheel artifacts will appear in `dist/`.
